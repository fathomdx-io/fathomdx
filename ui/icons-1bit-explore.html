Where these land

  • Claude Code mark — next to each session: chip in chat, and anywhere the dashboard lists claude-code sessions. Rendered at 14–16 px.
  • OS badges — dashboard host cards (Linux / macOS / Windows), and wherever a site surface attributes a delta to a host. 12–14 px inline; 20–24 px on detail cards.

The SVG in each "source" block above is the deliverable. Flat icons (Claude Code, macOS, Windows) use currentColor on the root — tint them by setting color: on a parent. Tux uses two tones: body paths carry currentColor (body opacity), accent paths (belly, feet, eye-whites) carry a hardcoded accent color (default white) — so its eye-whites stay white no matter what surface they're on.
